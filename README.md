# Rate limiter simulator

## Instalation

```
git clone git@github.com:dusmel/rate-limiter-simulator.git
cd rate-limiter-simulator
pnpm install
pnpm dev
```

Default config


```
  timewindow: 10 seconds, max: 4 requests per client
  monthly: 30 days, max 6 requests per client
  system: 10 requests 
```
